# Hello World This is an example of the Markdown Rendering in GitHub.dart! ## Usage In your Dart Code: ```dart import "package:github/browser.dart"; void main() { var github = createGitHubClient(); // Renders Elements with the markdown attribute GitHubBrowserHelper.renderMarkdown(github, "*[markdown]"); } ``` In your HTML: ```html <div hidden markdown> # Your Markdown Here </div> ```
